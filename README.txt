# Long Distance Site — Version 6 (Hosting Ready)

This is the exact v6 you liked (taller photos, simple design). It includes configs for **Vercel** and **Netlify**.

## One‑click-ish deploy options

### Option A — Netlify (fastest, drag & drop)
1) Go to https://app.netlify.com/drop
2) Drag this entire folder in. It will publish at a URL like `https://mokka-and-__.netlify.app`.
3) Click **Site settings → Domain management** to change to something cute, e.g. `mokka-baby.netlify.app`.
4) To use your own domain (e.g. `mokka.love`): add domain in Netlify, then update DNS to the Netlify records they show.

### Option B — Vercel
1) Install the CLI: `npm i -g vercel`
2) In this folder, run: `vercel --prod`
3) Rename the project to a cute name; it will deploy to `https://<name>.vercel.app`.
4) Add a custom domain in the Vercel dashboard if desired.
